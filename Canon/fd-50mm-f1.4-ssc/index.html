    Canon FD 50mm f/1.4 SSC

    A sharp, fast standard lens from Canon's FD mount era (1971–1987). The SSC ("Super Spectra Coating") version is the most desirable variant. Common service issues are fungus or haze on internal elements and oil migration onto the aperture blades.

    Specifications

    Spec Value
    Focal length 50mm
    Max aperture f/1.4
    Min aperture f/16
    Elements / Groups 7 / 6
    Aperture blades 8
    Min focus distance 0.6m
    Filter thread 52mm
    Mount Canon FD

    Tools & Materials

    • JIS screwdrivers: #0, #1
    • Spanner wrench (adjustable)
    • Rubber friction tools (lens cap openers work well)
    • Sensor swabs and lens tissue
    • 99% isopropyl alcohol
    • Lighter fluid (Ronsonol) — for dissolving old grease
    • Helicoil grease (e.g. Molykote EM-30L or Nyogel 767A)
    • Aperture blade oil remover (lighter fluid on a cotton bud)
    • Cotton gloves
    • Cleaning solution (Eclipse or Zeiss wipes for elements)

    Service Steps

    Step 1: Assess the lens

    Before disassembly, check the lens wide open on a light source. Note any haze, fungus, or dust. Stop down and confirm the aperture blades move freely and are oil-free.

    Step 2: Remove the rear mount

    Using a JIS #1 screwdriver, remove the three screws securing the rear bayonet mount. Lift the mount off. The chrome aperture ring is now accessible.

    JIS vs Phillips

    These screws are JIS, not Phillips. Using the wrong driver will strip them. JIS screwdrivers have a slightly different tip angle.

    Step 3: Remove the aperture ring and rear group

    Unscrew the rear element group (it threads out counterclockwise). Set it aside on a clean surface element-side up. The aperture linkage arm will be visible once the rear group is removed.

    Step 4: Clean aperture blades (if oily)

    If blades have oil on them, remove the aperture assembly. Dip a cotton bud in lighter fluid and gently wipe each blade. Let fully dry before reassembly. Blades should be completely matte black — any sheen indicates remaining oil.

    Do not use excessive solvent

    Too much solvent can seep behind elements or dissolve lacquered surfaces. Use a barely-damp bud.

    Step 5: Clean optical elements

    Use lens tissue or a sensor swab with a drop of Eclipse fluid. Clean in circular motions from center outward. Inspect with a loupe. Repeat until clear.

    Step 6: Regrease the helicoid

    Remove the focus ring by unscrewing the set screw (under a cosmetic ring at the front). Separate the helicoids. Note the engagement point before separating — mark it with a marker if needed.

    Clean all old grease with lighter fluid. Apply a thin coat of helicoid grease to the threads. Reassemble at the correct engagement point.

    Engagement point

    The helicoid must be re-engaged at the same thread start it came apart at, or focus infinity will be off. Mark the thread start with a Sharpie before separating.

    Step 7: Reassemble and verify

    Reassemble in reverse order. Check:

    • [ ] Aperture blades snap cleanly at all stops
    • [ ] Focus throw feels smooth with no tight spots
    • [ ] No haze or cleaning marks visible on elements
    • [ ] Rear mount screws are snug
    • [ ] Infinity focus is accurate (check on a distant subject)

    Notes

    • The SSC version has a chrome ring near the rear; the later S.C. version does not have this ring.
    • If infinity is slightly off after reassembly, the rear element group can be shimmed by rotating it slightly in its thread — very small adjustments have a noticeable effect.
